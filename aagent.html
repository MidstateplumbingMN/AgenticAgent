security
Password hashing: SHA-256 with random per-installation salt. In-browser only, never transmitted. Brute-force: 5 attempts → 30s lockout (doubles each lockout).
google sheets
Token security: OAuth access token lives in-memory only — never written to localStorage or sessionStorage. Sheet ID stored in sessionStorage (cleared on browser close). Client ID safe to persist (public-facing by design).
